# 🏆 NIS ID — Общешкольная Цифровая Экосистема Рейтинга

**NIS ID** — это полноценная Fullstack-система электронного учета достижений, верификации учеников и автоматизации школьного рейтинга ("Зал Славы"), разработанная для экосистемы Назарбаев Интеллектуальных Школ. 

Проект масштабирован из узкоспециализированного спортивного модуля в единую цифровую платформу для трансляции активности каждого учащегося школы.

## 🚀 Архитектура и Технологический стек

* **Backend:** Язык **Go (Golang)**, использование стандартной библиотеки `net/http` для создания легковесного и быстрого REST API.
* **Database:** Реляционная СУБД **SQLite**. Логика связей "Один ко многим" реализована через `FOREIGN KEY` с каскадным удалением данных для обеспечения целостности СУБД.
* **Frontend:** Динамический Dashboard на **HTML5, CSS3 (Grid/Flexbox Layout) и JavaScript (Async/Await Fetch API)** с автоматическим рендерингом лидеров на пьедестале.
* **Интеграция:** Внешнее публичное API для генерации персональных QR-кодов верификации без скачивания тяжелых сторонних библиотек.

## 🛠️ Реализованный функционал

1. **Реляционный учет данных:** Раздельное управление цифровыми профилями учеников (ФИО, Класс, Роль/Статус в школе) и связанной ведомостью их достижений (Спорт, Академия, Волонтерство).
2. **Динамический SQL-расчет:** Сервер на ходу агрегирует (`SUM`) баллы из разных таблиц базы данных и автоматически сортирует общую таблицу по убыванию рейтинга.
3. **Умные интерактивные фильтры:** Возможность мгновенной фильтрации всей школы по параллелям (классам) или направлениям деятельности в один клик без перезагрузки страницы.
4. **QR-Верификация и цифровой след:** Генерация персональных QR-кодов для бейджей. При сканировании система считывает параметры адресной строки и подсвечивает нужного ученика в базе данных.
5. **Защита от системных ограничений:** Запуск проекта переведен в режим интерпретации на лету (`go run`), что позволяет обходить блокировки локально скомпилированных файлов `.exe` со стороны Device Guard в школьной сети.

## 📦 Инструкция по локальному запуску

Запуск производится через командную строку Windows в режиме выполнения на лету:

1. Установите компилятор Go.
2. Склонируйте репозиторий и перейдите в папку проекта.
3. Запустите бэкенд-сервер командой:
```bash
   go run main.go
